# Forest Fire Detection System

## About the Project
This project predicts forest fire risk using machine learning models.  
It analyzes weather and location data to classify fire risk levels (Low, Medium, High).  
The goal is to assist environmental agencies in early detection and prevention.

## Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  

## Features
- Data preprocessing and cleaning  
- Model training and evaluation  
- Fire risk prediction  
- Visualization of results
